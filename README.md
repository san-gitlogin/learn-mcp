# Learn MCP — The Fun Way

An interactive, quiz-based course to learn the **Model Context Protocol (MCP)** without falling asleep reading documentation.

🔗 **Live Site:** [https://san-gitlogin.github.io/learn-mcp/](https://san-gitlogin.github.io/learn-mcp/)

## What's Inside

10 bite-sized lessons covering:

1. Prerequisites & Claude API basics
2. What is MCP (with analogies that stick)
3. The 3 Superpowers — Tools, Resources, Prompts
4. Building an MCP Server in Python
5. Building an MCP Client
6. Resources & Prompts
7. Sampling — reverse communication
8. Roots & Notifications
9. Transport — stdio vs HTTP
10. Final checkpoint with score

Each lesson includes interactive quizzes, real Python code from actual MCP projects, line-by-line explanations, and visual flow diagrams.
